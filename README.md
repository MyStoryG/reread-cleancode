실제 코드로 작성하면서 경험해보는 것이 주 목표입니다.

<h2>Rule</h2>

* 매주 토요일 23시 59분까지 코드 작성 완료
* 매주 일요일 오후 9시 전까지 리뷰 완료하기
* 매주 일요일 오후 9시 30분~ 10시 30분 정도 1시간 내외로 구글 meet 모임으로 학습 내용 공유하기
